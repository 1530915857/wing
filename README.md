# Wing

A minimal CSS framework.

## Download/Use

**Install with npm**

```sh
$ npm install wingcss
```

**Link to It**

```html
&lt;link rel="stylesheet" href="http://usewing.ml/dist/wing.min.css"/&gt;
```
