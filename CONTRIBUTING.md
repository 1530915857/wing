# Contributing

We love when people help us out, it makes Wing a better place.

## Getting Started

Fork the repo:

```sh
$ git clone https://github.com/KingPixil/wing.git
```

CD into the directory

```sh
$ cd wing
```

Open with a text editor

```sh
$ atom . || sublime .
```

Get coding!
