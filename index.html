Wing

Minimal CSS Framework

npm install wingcss
Is Wing for You?

Wing is made for those smaller side projects, as it isn't as full featured as Bootstrap, Foundation, etc. Wing styles basic elements, comes with a grid, and some fading animations. Still, what more do you need?


If you love Wing, then tweet it, star it, or fork it in the bottom right.

Documentation

How you use Wing.

Get Started
Run npm install wingcss, and link to ./node_modules/wingcss/dist/wing.min.css
Also, you can just add this to your <head>: <link rel="stylesheet" href="http://usewing.ml/dist/wing.min.css"
GRID

Wing is built with a grid. The grid is a 12-column fluid grid. The max-width is 960px. When rendered on smaller devices, it resizes accordingly. The code is minimal and responsive. In fact, try resizing your browser.

TYPOGRAPHY

Wing is built with responsive typography based on rems. Wing also changes everything to base 10, so 1 rem is 10px.

The font for headers, (h1, h2, h3, h4, h5, h6) is Quicksand. For paragraphs, it is Raleway.


This means that typography comes with things such as bold, italics, and underline.

BUTTONS

Wing comes with a variety of buttons, some simple, some a little more complex. Buttons come in four styles, outline, 3D, round, and clear.

Outline

This is the outlined button, a nice minimal button for every situation.

<button class="btn btn-outline">outline</button> - outlined button <button class="btn btn-outline">outline-inverted</button> - inverted colors outlined button
Round

This is the rounded button, an alternative to an outlined button.

<button class="btn btn-round">round</button> - round button <button class="btn btn-round">round-inverted</button> - inverted colors round button
