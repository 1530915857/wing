Wing

Minimal CSS Framework

About

Wing is a minimal CSS framework. It comes with

  • Base Reset
  • Typography
  • A Grid
  • Form Styles
  • Buttons
  • Animations
  • Utilities
  • To Infinity And Beyond...

Get Started

Link to Wing by simply doing one of the following:

  • Link from the Wing website, add this to your <head> tag: <link rel="stylesheet" href="http://wing.ml/dist/wing.min.css">
  • Run $ npm install wingcss, and link like: <link rel="stylesheet" href="./node_modules/dist/wing.min.css">
